# SoundTouch-JNI

Java bindings for [SoundTouch](https://www.surina.net/soundtouch). Can be used on both J2SE and Android.

SoundTouch-JNI version: 1.0.4  
SoundTouch version: 2.3.1

## Usage
docs: https://docs.tianscar.com/soundtouch-jni
