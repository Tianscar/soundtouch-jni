# soundtouch4j

JNI wrapper for SoundTouch. Can be used on both J2SE and Android.

SoundTouch version: 2.3.1  
SoundTouch's website: https://www.surina.net/soundtouch

## Usage
docs: https://docs.tianscar.com/soundtouch4j
